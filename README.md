# PM Skills Library for Claude

37 production-ready Claude skills for product managers, powered by frameworks from Shreyas Doshi, Lenny Rachitsky, Aakash Gupta, Teresa Torres, and more.

## Browse Skills

Visit the live site: **[pm-skills-for-claude](https://YOUR_USERNAME.github.io/pm-skills-for-claude/)**

## Quick Install

1. Download a skill `.zip` from the website
2. Unzip and move the folder into `.claude/skills/` in your project
3. Type `/{skill-name}` in Claude to activate it
